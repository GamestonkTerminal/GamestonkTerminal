# Crypto

## prints

> Last 15 Crypto Prints over last 24hrs.

```format
/crypto prints <coin>
```

**coin**

- The coin you want the prints of

**Examples**

```
/crypto prints coin:BTC-USD
```

---

## bigprints

> Last 15 Large Crypto Prints.

```format
/crypto bigprints <coin>
```

**coin**

- The coin you want the large prints of

**Examples**

```
/crypto bigprints coin:BTC-USD
```
