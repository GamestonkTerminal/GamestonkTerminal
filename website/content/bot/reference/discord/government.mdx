# Government

## contracts

> Displays Top 15 Government contracts by amount for all tickers or for a specific ticker.

```format
/gov contracts <ticker>
```

**ticker**
- The symbol of the stock you want to query

**Examples**

```
/gov contracts ticker:GE
```
