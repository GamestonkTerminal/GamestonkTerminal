# ETFs

## tops/holdings

> Displays ETF's Top Gainers/Decliners, Most Active or get holdings data for an ETF.

```format
/etf tops <sort>
/etf holdings <ticker>
```

**sort**
- Sort by Top Gainers, Top Decliners, or Most Active

**ticker**
- Ticker to get holdings data for

**Examples**

```
/etf holdings SPY
```

```
/etf tops
```
