# Due Diligence

## pm

> Displays premarket data for a given stock ticker

```format
/pm <ticker>
```

**ticker**

- The symbol/ticker of the stock you want the premarket info for

**Examples**

```
/pm AMD
```

---

## earnings

> Display Earnings that are expected to occur in the next business days. (Sorted by market cap)

```format
/earnings <day>
```

**day**

- The Date you want to see the earnings for. Format: YYYY-MM-DD

**Examples**

```
/earnings 2022-07-29
```
