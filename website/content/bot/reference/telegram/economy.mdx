## bonds

> Global bonds overview.

```format
/bonds <market>
```

**market** -
- The market to get bonds for - us (US bonds) or gl (Global)

**Examples**

```
/bonds us
```

```
/bonds gl
```

---

## revrepo

> Displays Reverse Repo.

```format
/revrepo [<days>]
```

**[days]** - _optional_
- The number of days to display. Default : 50

**Examples**

```
/revrepo days:35
```

```
/revrepo
```
