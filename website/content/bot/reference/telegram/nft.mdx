# NFT

## top

> Get Top NFT Collections from Ethereum

```format
/nft top [sortby] [interval] [reverse]
```

**[sortby]** - _optional_

- Column to sort by (default: Market Cap) - can be one of : Market Cap/ Floor Price Change / Volume / Sales

**[interval]** - _optional_

- Time interval to display Sales/Volume/Floor Price  (default: 1d) - can be one of : 1d / 7d / 30d / 90d

**[reverse]** - _optional_

- Reverse the sort order (default: No) - Yes/No

**Examples**

```
/nfttop
```

```
/nfttop mc y
```

## collection

> Get NFT collection data from Ethereum network for the specified slug. Returns ranking, price info, supply, and much more.

```format
/nftcollection <slug>
```

**slug**

- The nft you want the collection info for

**chart** -optional-

- returns a chart for the selected nft - can also use y/yes/chart/true

**Examples**

```
/nftcollection bent
```

```
/nftcollection bent chart
```

```
/nftcollection bent y
```
