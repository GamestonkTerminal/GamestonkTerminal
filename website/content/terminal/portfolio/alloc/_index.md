```
usage: alloc [-a {assets,sectors}] [-l LIMIT] [-t] [-h]
```

Show your allocation to each asset or sector compared to the benchmark.

```
optional arguments:
  -a {assets,sectors}, --agg {assets,sectors}
                        The type of allocation aggregation you wish to do (default: assets)
  -l LIMIT, --limit LIMIT
                        The amount of assets or sectors you wish to see (default: 10)
  -t, --tables          Whether to also include the assets/sectors tables of both the benchmark and the portfolio. (default: False)
  -h, --help            show this help message (default: False)

```
