End of the year returns

### Usage

```python wordwrap
yret [-p PERIOD]
```

---

## Parameters

| Name | Parameter | Description | Default | Optional | Choices |
| ---- | --------- | ----------- | ------- | -------- | ------- |
| period | -p  --period | Period to select start end of the year returns | all | True | 3y, 5y, 10y, all |
