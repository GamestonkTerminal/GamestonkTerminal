# export

Export dataset to Excel

### Usage

```python
usage: export [-t {xlsx,csv}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| type | The file type you wish to export to | xlsx | True | xlsx, csv |
