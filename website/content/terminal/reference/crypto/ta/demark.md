Calculates the Demark sequential indicator.

### Usage

```python wordwrap
demark [-m MIN_TO_SHOW]
```

---

## Parameters

| Name | Parameter | Description | Default | Optional | Choices |
| ---- | --------- | ----------- | ------- | -------- | ------- |
| min_to_show | -m  --min | Minimum value of indicator to show (declutters plot). | 5 | True | None |
