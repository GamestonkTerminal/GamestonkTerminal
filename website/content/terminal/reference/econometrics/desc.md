# desc

Show the descriptive statistics of the dataset

### Usage

```python
usage: desc -n {}
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| name | The name of the dataset.column you want to show the descriptive statistics | None | False | None |
