# rmv

Removes a price from the list

### Usage

```python
usage: rmv -p PRICE [-a]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| price | Price you want to remove from the list | None | False | None |
| all | Remove all prices from the list | False | True | None |
