# vsurf

Plot 3D volatility surface.

### Usage

```python
usage: vsurf [-z {IV,OI,LP}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| z | The data for the Z axis | IV | True | IV, OI, LP |
