# queries

Print top related queries with this stock's query. [Source: Google]

### Usage

```python
usage: queries [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | limit of top related queries to print. | 10 | True | None |
