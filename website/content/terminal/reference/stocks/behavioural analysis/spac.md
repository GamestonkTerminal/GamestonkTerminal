# spac

Show other users SPACs announcement. [Source: Reddit]

### Usage

```python
usage: spac [-l N_LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| n_limit | limit of posts with SPACs retrieved. | 5 | True | None |
