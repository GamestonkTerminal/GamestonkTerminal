# rot

Rating over time (monthly). [Source: Finnhub]

### Usage

```python
usage: rot [-l LIMIT] [--raw]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of last months | 10 | True | None |
| raw | Only output raw data | False | True | None |
