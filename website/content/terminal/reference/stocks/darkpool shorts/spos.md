# spos

Shows Net Short Vol. vs Position. [Source: Stockgrid]

### Usage

```python
usage: spos [-n NUM] [-r]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| num | Number of last open market days to show | 120 | True | None |
| raw | Flag to print raw data instead | False | True | None |
