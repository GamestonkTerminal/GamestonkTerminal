# gainers

Print up to 25 top gainers. [Source: Yahoo Finance]

### Usage

```python
usage: gainers [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
