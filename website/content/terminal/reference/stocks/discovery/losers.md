# losers

Print up to 25 top losers. [Source: Yahoo Finance]

### Usage

```python
usage: losers [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
