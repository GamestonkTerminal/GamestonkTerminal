# gtech

Print up to 25 top tech stocks with revenue and earnings growth in excess of 25%. [Source: Yahoo Finance]

### Usage

```python
usage: gtech [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
