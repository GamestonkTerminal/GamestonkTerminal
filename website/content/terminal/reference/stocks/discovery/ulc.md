# ulc

Print up to 25 potentially undervalued large cap stocks. [Source: Yahoo Finance]

### Usage

```python
usage: ulc [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
