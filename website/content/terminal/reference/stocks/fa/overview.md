Prints information about, among other things, the industry, sector exchange and company description.

### Usage

```python wordwrap
overview [-t TICKER]
```

---

## Parameters

| Name | Parameter | Description | Default | Optional | Choices |
| ---- | --------- | ----------- | ------- | -------- | ------- |
| ticker | -t  --ticker | Ticker to analyze | None | True | None |
