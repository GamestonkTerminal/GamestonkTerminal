# mktcap

Market Cap estimate over time. [Source: Yahoo Finance]

### Usage

```python
usage: mktcap [-s START]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| start | The starting date (format YYYY-MM-DD) of the market cap display | 2019-11-20 | True | None |
