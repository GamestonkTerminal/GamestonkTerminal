Plot multiple indicators on the same chart separated by a comma.

### Usage

```python wordwrap
multi -i INDICATORS
```

---

## Parameters

| Name | Parameter | Description | Default | Optional | Choices |
| ---- | --------- | ----------- | ------- | -------- | ------- |
| indicators | -i  --indicators | Indicators with optional arguments in the form of "macd[12,26,9],rsi,sma[20]" | None | False | None |
