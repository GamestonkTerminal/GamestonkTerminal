# sh

Provides the sharpe ratio of the selected stock.

### Usage

```python
usage: sh [-r RFR] [-w WINDOW]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| rfr | Risk free return | 0 | True | None |
| window | Rolling window length | 1 | True | None |
