# wfpe

Coin withdrawal fees per exchange [Source: https://withdrawalfees.com/]

### Usage

```python
usage: wfpe [-c COIN]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| coin | Coin to check withdrawal fees in long format (e.g., bitcoin, ethereum) | bitcoin | True | None |
