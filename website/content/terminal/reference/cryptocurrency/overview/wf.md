# wf

Display top coins withdrawal fees [Source: https://withdrawalfees.com/]

### Usage

```python
usage: wf [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit number of coins to display withdrawal fees. Default 10 | 10 | True | None |
