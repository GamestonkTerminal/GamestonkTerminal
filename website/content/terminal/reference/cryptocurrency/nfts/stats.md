# stats

Display stats about an opensea nft collection. e.g. alien-frens [Source: https://opensea.io]

### Usage

```python
usage: info -s SLUG
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| slug | Opensea collection slug (e.g., mutant-ape-yacht-club) | None | False | None |
