# stats

Display coin stats

### Usage

```python
usage: stats [--vs {USD,USDC,GBP,USDT,EUR,USD}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| vs | Quote currency (what to view coin vs) | USDT | True | USD, USDC, GBP, USDT, EUR, USD |
