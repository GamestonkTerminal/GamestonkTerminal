# stats

Display coin stats

### Usage

```python
usage: stats [--vs {USDT,EUR,USDC,USD,GBP,USD}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| vs | Quote currency (what to view coin vs) | USDT | True | USDT, EUR, USDC, USD, GBP, USD |
