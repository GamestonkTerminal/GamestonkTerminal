```text
usage: decompose [-m]
```

Decompose time series as:
 - Additive Time Series = Level + CyclicTrend + Residual + Seasonality
 - Multiplicative Time Series = Level * CyclicTrend * Residual * Seasonality

* -m : multiplicative model. Default additive.
