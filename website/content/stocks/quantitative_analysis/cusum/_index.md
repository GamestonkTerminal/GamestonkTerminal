```text
usage: cusum [-t THRESHOLD] [-d DRIFT]
```

Cumulative sum algorithm (CUSUM) to detect abrupt changes in data

* -t : threshold. Default (MAX-min)/40.
* -d : drift. Default (MAX-min)/80.
