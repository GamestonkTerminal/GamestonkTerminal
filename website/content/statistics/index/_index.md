```
usage: index [-n NAME [NAME ...]] [-i] [-h]
```

Set a (multi) index for a dataset, specified by `--name`. When setting two indices, it is assumed it follows the 
structure <entity> <time> (e.g. company names and quarter)

```
optional arguments:
  -n NAME [NAME ...], --name NAME [NAME ...]
                        The first argument is the name of the database, further arguments are the columns you wish to set as index (default: None)
  -i, --ignore          Whether to allow for making adjustments to the dataset to align it with the use case for Timeseries and Panel Data regressions (default: False)
  -h, --help            show this help message (default: False)
```
