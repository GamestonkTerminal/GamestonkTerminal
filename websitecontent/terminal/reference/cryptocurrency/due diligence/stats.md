# stats

Display coin stats

### Usage

```python
usage: stats [--vs {USD,GBP,USDT,USDC,USD,EUR}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| vs | Quote currency (what to view coin vs) | USDT | True | USD, GBP, USDT, USDC, USD, EUR |
