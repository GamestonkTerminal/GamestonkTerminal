usage: ftd [-s START] [-e END] [-n N_NUM] [--raw]

Prints latest fails-to-deliver data. [Source: SEC]

optional arguments:
  -s START, --start START
                        start of datetime to see FTD
  -e END, --end END     end of datetime to see FTD
  -n N_NUM, --num N_NUM
                        number of latest fails-to-deliver being printed
  --raw                 Print raw data.


Examples:
- To display the latest fails-to-deliver data for a specific stock: stocks/load <SYMBOL>/dps/ftd
- To show fails-to-deliver data for <SYMBOL> between specific dates: stocks/load <SYMBOL>/dps/ftd -s 2022-01-01 -e 2022-01-31
- To view the most recent 10 fails-to-deliver data points for a stock: stocks/load <SYMBOL>/dps/ftd -n 10
- Display raw fails-to-deliver data for a given stock: stocks/load <SYMBOL>/dps/ftd --raw
- For a specific stock, show fails-to-deliver data within a date range and limit the number of results: stocks/load <SYMBOL>/dps/ftd -s 2022-01-01 -e 2022-01-31 -n 5
