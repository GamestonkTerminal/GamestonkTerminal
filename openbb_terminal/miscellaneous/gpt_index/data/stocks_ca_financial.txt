parent_command:stocks/ca
usage: financial

Prints screener data of similar companies. [Source: Finviz]

financial
Examples:
- To display financial screener data for a specific company, first load the symbol and then use the financial command: stocks/load <SYMBOL>/ca/financial
- Display financial data for a company by loading its symbol and printing similar companies' screener data: stocks/load <SYMBOL>/ca/financial
- Load a stock symbol and retrieve financial screener data for similar companies: stocks/load <SYMBOL>/ca/financial
- To view financial screener data for companies similar to a specific stock, load the stock symbol and use the financial command: stocks/load <SYMBOL>/ca/financial
