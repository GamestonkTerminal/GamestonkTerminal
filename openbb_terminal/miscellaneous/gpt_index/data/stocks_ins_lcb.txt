parent_command:stocks/ins
usage: lcb [-l LIMIT]

Print latest cluster buys. [Source: OpenInsider]

optional arguments:
  -l LIMIT, --limit LIMIT
                        Limit of datarows to display (default: 10)


Examples:
- To display the latest cluster buys for a stock: stocks/ins/lcb
- To view the latest 5 cluster buys for a stock: stocks/ins/lcb -l 5
- To show the most recent 20 cluster buys for a stock: stocks/ins/lcb --limit 20
