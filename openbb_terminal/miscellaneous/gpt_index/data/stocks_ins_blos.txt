parent_command:stocks/ins
usage: blos [-l LIMIT]

Print latest officer sales 100k. [Source: OpenInsider]

optional arguments:
  -l LIMIT, --limit LIMIT
                        Limit of datarows to display (default: 10)


Examples:
- To display the latest officer sales 100k for a stock: stocks/load <SYMBOL>/ins/blos
- To display the latest officer sales 100k for a stock with a specific limit: stocks/load <SYMBOL>/ins/blos -l 5
- Show the latest 7 officer sales 100k for a stock: stocks/load <SYMBOL>/ins/blos --limit 7
- Retrieve the latest officer sales 100k for a stock and display only the top 3: stocks/load <SYMBOL>/ins/blos -l 3
