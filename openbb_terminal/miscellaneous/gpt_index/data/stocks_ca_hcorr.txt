usage: hcorr [-t {o,h,l,c,a}] [-s START] [-e END] [--display-full-matrix]

Correlation heatmap based on historical price comparison between similar companies.

optional arguments:
  -t {o,h,l,c,a}, --type {o,h,l,c,a}
                        Candle data to use: o-open, h-high, l-low, c-close, a-adjusted close, r-returns. (default: a)
  -s START, --start START
                        The starting date (format YYYY-MM-DD) of the stock (default: 2022-06-01)
  -e END, --end END     The end date (format YYYY-MM-DD) of the stocks (default: 2023-06-02)
  --display-full-matrix
                        Display all matrix values, rather than masking off half. (default: False)


Examples:
- To create a correlation heatmap for similar companies based on adjusted close prices: stocks/ca/hcorr
- Generate a correlation heatmap using high prices between specific dates: stocks/ca/hcorr -t h -s 2022-01-01 -e 2022-12-31
- Display a correlation heatmap for similar companies with low prices and a custom date range: stocks/ca/hcorr --type l --start 2021-07-01 --end 2022-07-01
- Create a correlation heatmap using open prices and display the full matrix: stocks/ca/hcorr -t o --display-full-matrix
- Generate a correlation heatmap based on returns between specific date ranges: stocks/ca/hcorr --type r --start 2021-01-01 --end 2022-01-01
- Display a correlation heatmap for close prices and show the complete matrix values: stocks/ca/hcorr -t c -s 2022-01-01 -e 2022-12-31 --display-full-matrix
